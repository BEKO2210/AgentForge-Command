# v0.1.0 — Initial public preview (MIT)

The first public release of the **4-Agent Team Kit for Claude Code**: a task-agnostic
scaffold that lets four Claude Code sessions build one repository together, coordinated
through plain files in a `.team/` folder and a handful of POSIX shell scripts.

> No database. No message broker. No framework. Four terminals, one repo, zero runtime
> dependencies in the core.

## Highlights

- **Atomic locking.** `scripts/lib/lock.sh` uses an atomic `mkdir` lock directory with
  PID-liveness stale detection and an atomic rename-based break, so two agents can
  never both acquire the same lock.
- **Logs are authority.** `team-sync.sh` folds the append-only per-agent logs into the
  state each board row *should* have and reports drift — without ever writing the board
  itself (the board has exactly one writer).
- **Liveness + recovery.** `team-health.sh`, `team-resume.sh`, `team-backup.sh`,
  `team-lead-claim.sh`, and `team-handoff.sh` cover stale tasks, deadlock signals,
  crash recovery, snapshots, fallback-lead, and cross-session handoffs.
- **Snapshots & diff.** `team-snapshot.sh` captures the whole state as one self-contained
  JSON; `team-diff.sh` reports what changed between two snapshots.
- **Optional GUI.** "TEAM // CONSOLE" runs all four sessions in one window with a live
  vitals strip (board progress + per-agent health). See the README screenshot.
- **Optional MCP server.** `mcp/` exposes the state as read-only MCP resources
  (`team://state | board | memory | …`) for Claude Desktop, IDE plugins, or any
  MCP-aware client.
- **Typed state contract.** `schema/team-state.schema.json` describes what the GUI,
  the MCP server, and snapshots agree on. `lib/state.mjs` is the single canonical
  implementation; the three consumers import it.
- **Tested in CI.** 87-check Bash test suite + 12 MCP smoke checks, plus `npm audit`
  on `gui/` and `mcp/` lockfiles. The gate runs `bash -n` + `shellcheck -S warning`
  + the suite on every push.

## Quickstart

```bash
git clone https://github.com/BEKO2210/4-Agent-Team-Kit-for-Claude-Code
cd 4-Agent-Team-Kit-for-Claude-Code
bash scripts/team-demo.sh         # 30-second self-running demo, no Claude Code needed
# Or, install the kit into another repo:
bash scripts/team-init.sh /path/to/your/repo
```
