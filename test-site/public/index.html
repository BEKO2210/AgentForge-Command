Vier Agenten, ein Repo, eine Webseite.

Diese Mini-Site demonstriert die 4-Agent-Team-Kit-Pipeline: Lead plant, Backend liefert die API, Frontend baut das Interface, Quality sichert den Smoke-Test ab.

Was hier läuft

  • Statisches Frontend

    Drei semantische HTML-Seiten, ein gemeinsames CSS, keine Build-Pipeline.

  • Mini-Backend

    Node-http-Server liefert HTML aus und beantwortet /api/hello.

  • Quality-Smoke

    Bash-Skript bootet den Server und prüft, ob alle Routen 200 antworten.

Live-Probe der API
